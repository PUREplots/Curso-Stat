Vamos a usar muestreo con probabilidades.

Para eso tenemos que saber la probabilidad de seleccionar cualquier fila (cualquier subconjunto) de antemano. Necesitamos un plan para predecir lo que va a pasar.

Un ejemplo:

Tengo 3 colores: R, V y A. Quiero escoger 2 siguiedo algunas reglas: 

    1* Siempre eligo el A primero.
    2* Luego eligo R o V con la misma proba. 


    P(A R) = 1/2
    P(A V) = 1/2

 Pero tambin necesito describir la proba de escoger cualquiera de los otros subconjuntos:

 P(V A) = 0
 P(A) = 1
 P(V) = P(R) = 1/2
 P(A V R) = 0
